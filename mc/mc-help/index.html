Minecraft服务器指南

为了照顾对于Minecraft Java一点也不熟悉的人，这个页面可能会帮到你，简单来说，配置Mc java版只需要三步：
安装 -> 配置 -> 启动

①

安装Oracle Java™ 8 & 16 及更高版本

对于Java™版本，可以点击这一篇专栏查看-->各Minecraft常见版本所需Java版本及其常见答疑（尽量长期维护）


B站专栏跳转 下载Java8 下载Java16及更高版本
②

下载游戏启动器

游戏启动器有多种，这里推荐HMCL(Hello Minecraft! Launcher)和PCL II，唯一美中不足的是你需要自己去配置，没有现成的安装包


HMCL(Hello Minecraft! Launcher)

下载好HMCL后，请在D盘创建一个文件夹，将hmcl.jar(或是hmcl.exe)文件放进去
随后：鼠标右键hmcl文件-->(windows11用户需要展开完整的右键菜单)-->发送到-->桌面快捷方式-->（非必要性）退回桌面，给快捷方式改名


PCL II

下载好PCL2后，请在D盘创建一个文件夹，将下载好的压缩包zip文件放进去
随后：鼠标右键zip文件-->解压缩-->找到文件，移动到你创建的文件的起始点（比如说你在D盘创建了mc文件夹，其起始点就在 D:\mc\ ，也就是该文件夹的根目录）-->（非必要性）清理空文件夹-->鼠标右键Plain Craft Launcher 2.exe文件-->(windows11用户需要展开完整的右键菜单)-->发送到-->桌面快捷方式-->（非必要性）退回桌面，给快捷方式改名


下载HMCL启动器 下载PCL II启动器
③

获取账号

现有三种登录方式：离线登录、第三方登录和正版登录

如果预算充足，还是建议购买正版账号的

这里只讲第三方登录:LittleSkin

在官网注册好账号并且添加了角色之后，打开游戏启动器


1.HMCL

点击左上角添加账号，左边栏目选择LittleSkin，用户名填你注册时用到的邮箱地址，密码填注册时输入的密码


2.PCL II（若为常规顺序，该步骤暂时不可用）

在下载好MC游戏版本后，点击版本设置，左边栏目选择设置，往下翻找到服务器，第一个选择框选择“第三方登录：Authlib Injector 或 LittleSkin”，再点击按钮“设置为 LittleSkin”，然后返回主界面，填写你的账号消息，用户名填你注册时用到的邮箱地址，密码填注册时输入的密码


LittleSkin
④

下载对应版本

一定要在启动器设置里打开版本隔离！！！

Minecraft版本在Java版中分四种：正式版、快照版（预览版）、远古版和愚人节版，愚人节版比较特殊，你可以在PCL2中单独见到该分组，在HMCL中，你需要前往快照分组中找在4月1号发布的版本

首先点击“下载”，在一大堆Mc版本中选择你想玩的，正常来说是选择正式版

点进去后，选择模组加载器

目前最主流的模组加载器是Forge(铁砧图案)和Fabric(布料图案)，如果你选择Fabric，还需要安装Fabric API

注意，除Forge-1.20.1以外，Forge和Fabric是不兼容的
1.20.1可以安装信雅互联 | Sinytra connector模组来解决兼容性问题(游戏版本选择Forge，然后再安装此模组)

光影加载器是OptiFine，但与模组的兼容性极差

⑤

安装模组

两种方式，一种是在启动器内安装，一种是把模组拖入mods文件夹

要在启动器内安装请点击“下载”，找到模组栏目，现在MCMOD中的模组检索中找你想玩的模组，进入模组详情页后复制英文名到启动器的搜素框内，如果没搜到，那可能是模组没有被收录到CurseForge或Modrinth站点中

要把现有的jar或zip文件的模组（近代版本一般都是.jar文件）拖入到mods文件夹，请在启动器中选择打开Mod文件夹，将文件移进去就行了

部分模组有兼容性问题，请留意模组站中模组详情页面中所描述的兼容性问题

Forge-1.20.1以外的版本需注意区分Forge和Fabric模组
